# Теория вероятностей
Предмет исследования вероятностей - **событие**
Есть 3 группы событий: случайные, невозможные и достоверные
Случайные события характеризуются совместностью, зависимостью и противоположностью.
Классическая формула вероятностей: $P(A) = \frac{M}{N}$
Вероятность вычисляется теоретически до эксперимента, относительная частота события вычисляется по результатам экспериментов.

Сумма событий $A+B=C$: событие С происходит при **А или В**
Произведение событий $А*В=С$: событие С происходит при **А и В**

Закон распределения может задаваться таблицей, функцией плотности и функцией вероятности. Плотность является производной функции вероятности.

# Предварительный анализ данных
Включает:

- Числовые характеристики выборочной совокупности
- Исследование выборочной совокупности на наличие выбросов

## Сбор данных
Делится на:
- **Генеральную совокупность** (сплошной отбор) - исследование всех данных;
	- Активный сбор - самостоятельный сбор данных для конкретного исследования;
	- Пассивный сбор - использование ранее полученных опытных данных;
	- Простой (по одному) и не простой (группами);
	- Повторный (объект можно исследовать несколько раз) и безповторный;
- **Выборочная совокупность** (несплошной отбор) - исследование части данных.

Объекты исследования отбираются по разному:
- Типический сбор - выбор по типу (признаку);
- Серийный - извлечение нескольких объектов подряд (серии объектов);
- Механический - каждый $n$-й объект

Для того, чтобы судить генеральную совокупность по выборочной, необходимо, чтобы выборочная совокупность была **репрезентативной**:
- Полнота - должна представлять каждый из имеющихся признаков, т.е. быть достаточно полной;
- Системность - отображает системообразующие свойства выборки;
- Корректность - в процессе сбора данных не нарушаются правила эксперимента.

### Минимальный допустимый объем выборочной совокупности
- Повторный: $$n = \frac{t^2(\alpha, \infty)*D}{\Delta^2}$$
- Безповторный: $$n = \frac{t^2(\alpha, \infty)*D*N}{\Delta^2*N+t^2(\alpha, \infty)*D}$$

Где:
- $t^2(\alpha, \infty)$ - значение по таблице Стьюдента;
- $\alpha$ - вероятность ошибки;
- $N$ - объем генеральной совокупности;
- $D$ - дисперсия;
- $\Delta$ - предельная ошибка выборочной совокупности - насколько ожидаемо выборочное среднее отклонится от генерального среднего.

**Для доли:**
- Повторный: $$n = \frac{t^2(\alpha, \infty)*w(1-2)}{\Delta^2}$$
- Бесповторный: $$n = \frac{t^2(\alpha, \infty)*w(1-w)*N}{\Delta^2*N+t^2(\alpha, \infty)*w(1-w)}$$

Где $w$ - доля объектов выборочной совокупности, удовлетворяющих условию

### Величина предельной ошибки выборочной совокупности
**Для средней:**
- Повторный: $$n = t(\alpha, \infty)*\sqrt{\frac{D}{n}}$$
- Бесповторный: $$n = t(\alpha, \infty) * \sqrt{\frac{D}{N}*(1-\frac{n}{N})}$$

Где:
- $t(\alpha, ∞)$ — табличное значение статистики Стьюдента.
- $\alpha$ — вероятность совершения ошибки.
- $D$ — дисперсия
- $N$ — объем генеральной совокупности
- $n$ — объем выборочной совокупности

**Для доли:**
- Повторный: $$n = t(\alpha, \infty)*\sqrt{\frac{w(1-w)}{n}}$$
- Бесповторный: $$n = t(\alpha, \infty) * \sqrt{\frac{w(1-w)}{N}*(1-\frac{n}{N})}$$

Где:
- $w$ - доля объектов выборочной совокупности, удовлетворяющих условию

### Доверительный интервал
**Для среднего:**
$$x \in [\overline{x}-\Delta, \overline{x}+\Delta]$$
- Повторный: $$\Delta = t(\alpha, \infty)*\sqrt{\frac{D}{n}}$$
- Бесповторный: $$\Delta = t(\alpha, \infty) * \sqrt{\frac{D}{N}*(1-\frac{n}{N})}$$

**Для доли:** $$p \in [w-\Delta, w+\Delta]$$
- Повторный: $$\Delta = t(\alpha, \infty)*\sqrt{\frac{w(1-w)}{n}}$$
- Бесповторный: $$\Delta = t(\alpha, \infty) * \sqrt{\frac{w(1-w)}{N}*(1-\frac{n}{N})}$$
## Ряды распределения
**Вариационный ряд** - упорядочивание исходного ряда значений.
**Дискретный ряд** - ряд, полученный из вариационного путем удаления повторяющихся значений и указанием абсолютных и относительных частот появления значений ряда. Необходим для анализа равномерности частотности.
**Интервальный ряд** - ряд, полученный из исходного разбиением всего диапазона на интервалы и указания абсолютных и относительных частот попадания значений ряда в эти интервалы.

Формула Стёрджиса: $$\Delta = \frac{x_{max}-x_{min}}{1 + 3,32*lg(N)}$$
## Числовые характеристики выборочной совокупности
Делятся на:
- Характеристики центра группирования - вблизи какой величины преимущественно сгруппированны значения;
- Характеристики степени рассеивания - насколько в среднем отклоняются данные от центра группирования.
### Характеристики центра группирования
- **Среднее арифметическое** - характеризует наиборее ожидаемое значение выборочной совокупности.
  Взвешенное: $$\overline{x} = \frac{\sum_{i=1}^{N}x_if_i}{\sum_{i=1}^{N}f_i} \ \ \text{- когда распределение частотностей неравномерно}$$
  Невзвешенное: $$\overline{x} = \frac{1}{N}\sum_{i=1}^{N}x_i \ \ \text{- когда частотности равны}$$
- **Среднее квадратическое** - характеризует наиболее ожидаемое значение выборочной совокупности
  Взвешенное: $$\overline{x} = \sqrt{\frac{\sum_{i=1}^{N}x_i^2f_i}{\sum_{i=1}^{N}f_i}} \ \ \text{- когда распределение частотностей неравномерно}$$
  Невзвешенное: $$\overline{x} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}x_i^2} \ \ \text{- когда частотности равны}$$
- **Среднее геометрическое** - характеризует наиболее ожидаемое значение выборочной совокупности
  Взвешенное: $$\overline{x}=\sqrt[\sum_{i=1}^Nf_i]{\prod_{i=1}^Nx_i^{f_i}}$$
  Невзвешенное: $$\overline{x}=\sqrt[N]{\prod_{i=1}^Nx_i}$$
